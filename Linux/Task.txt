Задание 1
 
1)   Используйте команды операционной системы Linux и создайте две новых директории – 
     «Игрушки для школьников» и «Игрушки для дошколят»
2)   Создайте в директории «Игрушки для школьников» текстовые файлы - 
     «Роботы», «Конструктор», «Настольные игры»
3)   Создайте в директории «Игрушки для дошколят» текстовые файлы «Мягкие игрушки», «Куклы», «Машинки»
4)   Объединить 2 директории в одну «Имя Игрушки»
5)   Переименовать директорию «Имя Игрушки» в «Игрушки»
6)   Просмотреть содержимое каталога «Игрушки».
7)   Установить и удалить snap-пакет. (Любой, какой хотите)
8)   Добавить произвольную задачу для выполнения каждые 3 минуты 
     (Например, запись в текстовый файл чего-то или копирование из каталога А в каталог Б).


-------------Выполнение------------------------------------------

1) mkdir 'Игрушки для школьников'
   mkdir "Игрушки для дошколят"         Создание дирекоторий

2) cd Игрушки\ для\ школьников/         Переход в директорию "Игрушки для школьников"
   cat > Роботы.txt                     
   cat > Конструктор.txt                Создание текстовых файлов
   cat > "Настольны игры".txt 

3) cd ..                               Переход на одну директорию назад
   cd Игрушки\ для\ дошколят/          Переход в директорию "Игрушки для дошколят"
   cat > "Мягкие игрушки".txt
   cat > Куклы.txt                     Создание текстовых файлов
   cat > Машинки.txt

4) mkdir "Имя Игрушки"
   cp -r Игрушки\ для\ школьников/* Имя\ Игрушки/  
   cp -r Игрушки\ для\ дошколят/* Имя\ Игрушки/   Объединение двух директорий в одну "Имя Игрушки"

5) mv Имя\ Игрушки/ Игрушки            Переименование директории "Имя Игрушки" в "Игрушки"

6) cd Игрушки
   ll или ls или ls -a или ls -al      Просмотр содержимого директории "Игрушки"

7) sudo snap install chromium          Установка snap-пакета chromium

8) crontab -e - вход в редактор задач
   */3 * * * * date > ~/date.txt - в редакторе создал задачу записывать в файл каждые три минуты дату
